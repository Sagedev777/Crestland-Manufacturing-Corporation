JavaScript Exercises 1-5

1. Hello World with User Input

Create a function that asks for the user's name and displays a greeting.

2. Simple Calculator

Create a calculator that can add, subtract, multiply and divide two numbers.

3. Even or Odd Checker

Write a function that checks if a number is even or odd.

4. Temperature Converter

Create functions to convert between Celsius and Fahrenheit.

Formula: °F = (°C × 9/5) + 32 | °C = (°F - 32) × 5/9

5. String Reverser

Write a function that reverses a string without using built-in reverse() method.
